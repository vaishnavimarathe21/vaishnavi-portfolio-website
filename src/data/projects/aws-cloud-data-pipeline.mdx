# AWS Cloud Data Pipeline

## Project Overview

This project showcases an end-to-end AWS data engineering solution for analyzing global YouTube trending video data. A scalable, cloud-native pipeline was built to ingest, process, and analyze large volumes of structured and semi-structured data using AWS services.

## Architecture & Services

### Data Lake Architecture
- **Amazon S3** - Scalable object storage for raw and processed data
- **AWS Glue** - Serverless ETL service for data transformation
- **AWS Lambda** - Event-driven compute for data processing
- **AWS Athena** - Interactive query service for S3 data
- **QuickSight** - Business intelligence and visualization platform

### Data Flow
1. **Data Ingestion** - YouTube trending videos from multiple regions
2. **Data Processing** - ETL transformations using AWS Glue
3. **Data Storage** - Structured storage in S3 data lake
4. **Data Analysis** - SQL queries using AWS Athena
5. **Data Visualization** - Dynamic dashboards in QuickSight

## Key Achievements

### Performance Metrics
- **200,000+ Records** - Processed YouTube trending videos
- **90% Data Redundancy Reduction** - Through optimized ETL processes
- **60% Faster Query Performance** - With optimized data structures
- **Multi-Region Analysis** - Data from 10+ geographic regions

### Technical Implementation
- **Serverless Architecture** - Cost-effective and scalable solution
- **Data Lake Design** - Centralized repository for multiple data sources
- **ETL Optimization** - Efficient data transformation pipelines
- **Real-time Analytics** - Dynamic dashboards for business insights

## Dataset Details

### Source Data
- **YouTube Trending Videos Dataset** - Daily popular videos from multiple countries
- **200 Videos per Day** - Up to 200 trending videos per region
- **Multiple Months** - Historical data spanning several months
- **Rich Metadata** - Video titles, channels, views, likes, comments, categories

### Data Fields
- Video title and channel information
- Publication time and duration
- View counts, likes, dislikes
- Comment counts and engagement metrics
- Category IDs and regional data
- Tags and descriptions
